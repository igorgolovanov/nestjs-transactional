# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`, a build tool that works with multi-package repos, or single-package repos to help you version and publish your code.

You can find the full documentation for it [in our repository](https://github.com/changesets/changesets).

We have a quick list of common questions to get you started engaging with this project in [our documentation](https://github.com/changesets/changesets/blob/main/docs/common-questions.md).

## Usage in this repo

For any user-facing change to one of the published packages (`@nestjs-transactional/core`, `@nestjs-transactional/typeorm`, `@nestjs-transactional/cqrs`):

```bash
pnpm changeset
```

Follow the prompts: pick the packages affected, pick the bump (`patch` / `minor` / `major`), write a short description. The resulting `.md` file is committed alongside the code change.

At release time, `pnpm changeset:version` consolidates pending changesets into version bumps and `CHANGELOG.md` entries, and `pnpm changeset:publish` runs `pnpm build && changeset publish` to push to npm.
